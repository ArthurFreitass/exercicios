HTML tags:

A tag a:blank, me permite adicionar um link externo na minha página.

Outra tag muito utilizada é a tag A , pois permite a utilização de links internos, ou seja ele permite que eu acesse uma parte da minha página, apenas clicando nela. Por isso sendo de grande ajuda para a utilização do usuário no site. Basicamente a tag A, nos permite utilizar links internos, através do id="Alguma coisa", e do href = "#Alguma coisa" note que as dois precisam ser iguais para funcionar.
